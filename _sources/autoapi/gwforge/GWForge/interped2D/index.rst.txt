   A 2D interpolated Prior that samples from a joint PDF over two variables.

   :param name: Name of the prior (e.g., 'mass_pair')
   :type name: str
   :param x_axis: 1D array of values for the first variable
   :type x_axis: array-like
   :param y_axis: 1D array of values for the second variable
   :type y_axis: array-like
   :param pdf_grid: 2D grid of probability densities with shape (len(x_axis), len(y_axis))
   :type pdf_grid: 2D array
   :param x_name: Name of first variable (for labels, default 'x')
   :type x_name: str, optional
   :param y_name: Name of second variable (for labels, default 'y')
   :type y_name: str, optional
